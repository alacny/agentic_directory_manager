This is a simple toy-tool for creating and manipulating a directory structure with Artificial Intelligence. It is in a very early stage and not yet production ready. It is in Polish but you can describe actions in English, for example the prompt:

Hello, Create, please for me a directory structure containing names of noblists who won their prices on astronomy and containing subdirectory names when and for what they got it.

resulted with a following directory structure:

Nobelists_Astronomy
    ├── Andrea_Ghez
    │   └── 2020
    ├── Arno_Penzias
    │   └── 1978
    ├── Reinhard_Genzel
    │   └── 2020
    ├── Robert_Wilson
    │   └── 1978
    ├── Roger_Penrose
    │   └── 2020
    └── Subrahmanyan_Chandrasekhar
        └── 1983
